LuxuryVoyage – Explore Beyond Dreams

Project Description

LuxuryVoyage is a modern and responsive travel website developed using HTML, CSS, and JavaScript. The website features a luxury black theme with glitter pink accents, smooth animations, and a user-friendly interface for exploring travel destinations and packages.

Features

- Responsive Navigation Bar
- Mobile Hamburger Menu
- Hero Section
- Popular Destinations
- Why Choose Us Section
- Travel Packages
- Customer Reviews
- Contact Form
- JavaScript Contact Form Alert
- Smooth Scrolling
- Hover Effects and Animations
- Fully Responsive Design for Desktop, Laptop, Tablet, and Mobile Devices
- CSS Media Queries for Different Screen Sizes

Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome Icons
- Google Fonts (Poppins)

Responsive Design

This website is fully responsive and automatically adapts to different screen sizes.

- Desktop & Laptop: Full navigation bar with a multi-column layout.
- Tablet: Responsive grid layout with optimized spacing.
- Mobile: Hamburger menu, single-column layout, responsive images, and CSS media queries for a smooth user experience.
